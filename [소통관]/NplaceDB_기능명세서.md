## 1. 프로그램 개요
본 프로그램은 네이버 플레이스(Naver Place)의 업체 정보를 자동으로 수집하여 마케팅 DB로 변환해주는 전문가용 솔루션입니다. 특히 소상공인 및 기업의 영업 리스트 확보를 목표로 하며, Playwright 기반의 고도화된 크롤링 엔진을 탑재하고 있습니다.

## 2. 핵심 UI 기능 (gui_main.py, auth_gui.py)
- **인증 화면 (Auth Flow)**:
    - Supabase 기반의 실시간 라이선스 검증 시스템.
    - HWID(기기 고유 식별자) 바인딩을 통한 1PC 1라이선스 원칙 구현.
    - 테스트용 키(TEST-) 감지 시 수집 건수 및 기간 제한 팝업 안내.
- **메인 대시보드 (Premium Dark Mode)**:
    - 키워드 및 지역(동 단위 정밀 수집 지원) 입력 필드.
    - 제외 키워드 필터링 기능 (블랙리스트 기반).
    - 실시간 진행 상황 카드: 예상 대상 수, 현재 수집 개수, 소요 시간, 예상 남은 시간(ETA).
    - 시각적인 진행률 바(Progress Bar) 및 하단 상태 브리핑.
- **로그 및 데이터 뷰**:
    - **작업 로그 탭**: 엔진의 작동 상태, 에러 상황, 휴식 시간 등을 실시간으로 출력.
    - **수집 데이터 탭**: SQLite에 저장된 최신 50건의 데이터를 즉시 미리보기 할 수 있는 스크롤 뷰.

## 3. 수집 엔진 (step1_refined_crawler.py)
- **크롤링 로직 (Backend)**:
    - **Playwright + Stealth**: 봇 탐지 우회를 위한 스텔스 모드 적용.
    - **Apollo State 추출**: HTML 내부에 포함된 JSON 데이터(`__APOLLO_STATE__`)를 직접 파싱하여 가장 정확한 정보를 수집.
    - **풍부한 정보 확보**: 상호명, 전화번호, 주소뿐만 아니라 이메일(mailto/regex), 인스타그램 핸들, 네이버 블로그 ID, 톡톡 링크 등을 수집.
    - **Smart Skip**: 로컬 DB를 참조하여 이미 수집된 URL은 중복 수집하지 않고 즉시 건너뛰기 수행.
    - **휴먼 시뮬레이션**: 랜덤한 지연 시간(Sleep), 마우스 스크롤 시뮬레이션, User-Agent 로테이션을 통해 차단 방지.

## 4. 기타 도구
- **빌드 시스템**: `build.bat`, `PyInstaller`를 이용한 DLL 번들링 방식의 '무설치' 배포판 제작.
- **업데이트 엔진 (updater.py)**: 실행 시 서버와 통신하여 최신 버전을 감지하고 자동 다운로드 및 패치를 수행.
- **데이터 내보내기 (exporter.py)**: 수집된 DB를 즉시 Excel(`.xlsx`) 형식으로 정렬하여 저장.
